# الغزو الثقافي يمتد في فراغنا — Cultural Invasion Spreads into Our Void

Interactive trilingual web app based on **Sheikh Mohammed al-Ghazali**'s book (Dar Al-Shorouk, 1985).

## About the Book

Sheikh al-Ghazali diagnoses the diseases of the Muslim world, emphasizing that internal causes are more dangerous than external aggression. He covers: intellectual void, blind following, superficial understanding of religion, extremism and liberalism, and how cultural enemies fill the void left by Muslims.

## Features

- **20 Shield Cards**: 10 threats (internal/external) + 10 defenses
- **"Spot the Influence" Quiz**: 10 scenario-based questions with lifelines (50/50, hint, verse)
- **XP & Badges**: Earn points for reading cards and completing quizzes
- **Trilingual**: Arabic, English, French
- **3 Themes**: Fortress (light slate), Night (dark), Steel (neutral)
- **Audio Narrator**: Read page, read as book, karaoke, speed/pitch controls, duo reading
- **Daily Shield Card**: Rotating daily lesson
- **Streak Tracking**: Daily engagement streaks
- **PWA Support**: Installable on mobile

## Author

**Sheikh Mohammed al-Ghazali** (1917-1996) — Egyptian Islamic scholar and thinker, author of 94+ books.

## Tech

Pure HTML/CSS/JS, no frameworks. Web Audio API for sound effects. Web Speech API for narrator. localStorage for persistence.
